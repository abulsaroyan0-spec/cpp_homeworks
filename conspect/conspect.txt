Shell-ը ծրագրային շերտ է, որը հանդիսանում է միջնորդ օգտագործողի և օպերացիոն համակարգի (մասնավորապես՝ kernel-ի) միջև։

 Shell-ը հրամանների ինտերպրետատոր է (command interpreter)։
Դու գրում ես հրաման → shell-ը այն վերլուծում է → կանչում է համապատասխան system call-երը → kernel-ը կատարում է։

Գործառույթներ

Կարդում է հրամաններ (stdin)

Parse է անում (tokenization, expansion)

Կատարում է գործընթացներ (fork(), exec())

Աջակցում է redirection (>, <, >>)

Աջակցում է pipe (|)

Աջակցում է scripting (bash script)

Ամենահայտնի shell-երը

Bash – Linux-ում default

Zsh – ավելի հզոր interactive միջավայր

Fish – beginner-friendly

PowerShell – Windows-ում



🖥 Ինչ է Տերմինալը (Terminal)

Տերմինալը ծրագիր է, որը բացում է shell session։

👉 Տերմինալ = UI միջավայր
👉 Shell = հրամանների մշակող

Օրինակ՝

GNOME Terminal

iTerm2

Windows Terminal

Տերմինալը պարզապես emulator է, որը

ապահովում է stdin/stdout

ցուցադրում է output

կառավարում է pseudo-tty

Shell-ը ներսում է աշխատում։


📚 Տերմինալի Բազային Հրամաններ (Unix/Linux)

📁 Ֆայլային համակարգ
pwd
Ցույց է տալիս current working directory։

ls
Ցույց է տալիս directory-ի պարունակությունը։
ls
ls -l
ls -a
-l → long format
-a → hidden ֆայլեր

cd
Փոխում է current directory։
cd folder
cd ..
cd ~

mkdir
Ստեղծում է directory։
mkdir test
rmdir
Ջնջում է դատարկ directory։
rmdir test

rm
Ջնջում է ֆայլ։
rm file.txt
rm -r folder
⚠️ Վտանգավոր հրաման է։


📄 Ֆայլերի հետ աշխատանք
touch
Ստեղծում է դատարկ ֆայլ։
touch a.txt

cp
Copy է անում ֆայլ։
cp a.txt b.txt

mv
Move կամ rename է անում։
mv a.txt new.txt

cat
Տպում է ֆայլի պարունակությունը։
cat file.txt

less
Բացում է ֆայլ paging ռեժիմով։
less file.txt

🔎 Որոնում
grep
Փնտրում է pattern։
grep "main" file.c

find
Փնտրում է ֆայլ համակարգում։
find . -name "a.txt"
